## Расширение:
+ Подсветка синтаксиса:

  - Специальные символы
+ Диагностика и автодополнение:
  - Нахождение и подчеркивание строки, не содержащей ```;```

+ Сниппеты:
  - Для конструкций объявления пользовательских операторов, отношений и унификаций.
